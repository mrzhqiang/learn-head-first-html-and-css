## 本地预览？
- 安装 [开发环境][2]
- 安装 [Jekyll][1]
```bash
$ gem install jekyll bundler
```
- 安装主题
```bash
$ gem install just-the-docs
```
- 开启本地预览
```bash
$ bundle exec jekyll serve
```
- 打开浏览器并访问 [http://localhost:4000](http://localhost:4000)
